Projects

Home Lab - WAF - Web Security
SafeLine WAF - Web Attack Detection and Blocking Lab
Read Full Report GitHub

Deployed a fully isolated home lab using DVWA on an Ubuntu Server VM and a Kali Linux attacker VM to simulate a real threat environment. Executed live attacks including SQL Injection, XSS, and HTTP Flood, then deployed SafeLine WAF (Docker-based) as a reverse proxy in front of DVWA.

Configured security rules including rate limiting, IP blocking, authentication enforcement, and SQL injection/XSS detection. Verified every attack was blocked via 403 responses and real-time log entries showing source IP, matched rule, payload, and timestamp - directly mirroring SOC web traffic investigation workflows.

Home Lab - Firewall - Network Defense
pfSense Firewall Home Lab - SYN Flood Attack Simulation and Blocking
Read Full Report GitHub

Built a fully isolated dual-network firewall environment from scratch using pfSense 2.8.1 on VirtualBox with a Kali Linux attacker VM on the WAN side and an Ubuntu management VM on the LAN side. Configured pfSense via console, set up WAN/LAN interfaces, enabled DHCP, and established connectivity across both networks.

Launched a real SYN flood attack using hping3 (~30,000 packets/sec) and captured it live in Wireshark. Created custom WAN firewall rules to block the attacker IP and SYN flood packets - verified full blocking via pfSense firewall logs. Directly mirrors perimeter defense, DoS mitigation, and log analysis workflows used in SOC environments.
