Home Lab - SIEM - Detection Engineering
Wazuh Enterprise SIEM Lab - 6-Day SOC Simulation
GitHub

Deployed a 3-machine Wazuh SIEM lab (Ubuntu manager + Windows 11 + Kali Linux agents) and wrote 5 custom detection rules mapped to MITRE ATT&CK — covering PowerShell obfuscation (T1059.001), download cradles (T1105), Office macro shell spawning (T1566.001), persistence via scheduled tasks (T1136.001), and frequency-based brute force correlation (T1110). Simulated every attack from Kali and validated each rule fired with real alert evidence.

Ran a full incident response exercise — reconstructed a multi-stage attack chain from SIEM alerts, performed agent isolation, eradication, and produced a formal IR report (IR-2026-001). Integrated VirusTotal API for automated file hash enrichment and configured Active Response for automatic IP blocking on brute force triggers.

Home Lab - WAF - Web Security
SafeLine WAF - Web Attack Detection and Blocking Lab
Read Full Report GitHub

Deployed DVWA on Ubuntu and used Kali Linux to execute live SQL Injection, XSS, and HTTP Flood attacks — confirmed exploitable without protection. Then placed SafeLine WAF (Docker-based reverse proxy) in front of the application and re-ran every attack.

All attacks blocked. Real-time dashboard logs showed source IP, matched rule, payload, and timestamp for each blocked request — the same alert triage workflow SOC analysts run when investigating suspicious web traffic.

Home Lab - Firewall - Network Defense
pfSense Firewall Home Lab - SYN Flood Attack Simulation and Blocking
Read Full Report GitHub

Built a dual-network lab with pfSense 2.8.1 as the perimeter firewall — Kali Linux on WAN as the attacker, Ubuntu on LAN as the protected host. Launched a SYN flood attack via hping3 at ~30,000 packets/sec and confirmed impact with Wireshark before any rules were in place.

Created custom WAN firewall rules to block the attacker IP and SYN flood traffic — re-ran the attack and verified full blocking via pfSense firewall logs. Covers perimeter defense, DoS mitigation, and the log review cycle that mirrors real SOC triage.
